## PrepPal Vue App

This project now uses a Vue 3 CDN app with separated layout files.

- `index.html` — login page and login script
- `styles.css` — shared app styles
- `sidebar.html` — sidebar markup loaded as a separate component
- `app.js` — login logic for `index.html`
- `dashboard.html` — standalone dashboard page loaded after login
- `dashboard.js` — dashboard page app logic

Open `index.html` in a browser to run the login flow; successful login redirects to `dashboard.html`.
