What is PrepPal?

🎯 The Vision

PrepPal addresses the friction in modern student revision. Rather than jumping between separate planning, testing, and note-taking apps, PrepPal integrates progress metrics directly with learning content to enable active recall and structured scheduling in one central hub.

👥 Role-Based Experience

The platform is customized for three user tiers: • Students: Revise via decks, solve quizzes, track skill gaps, book tutorships. • Tutors: Host sessions, verify rates, create quizzes, customize resources. • Admins: Moderate content flags, publish global alerts, curate users.

🔄 Active Learning Loop

Every quiz attempt and flashcard study session updates the student's personal analytics profile. If a skill gap is detected, the automated planner recommends custom study tasks, closing the gap between study and assessment.

⚡ Modern Architecture

Designed to run fast and light. A decoupled backend serving a single-page interactive UI via modular components ensures sub-second page loads and seamless dashboard operations.

Deployment with Vercel

⚡

Serverless Integration

By exporting the Express app module and configuring rewrites, Vercel automatically deploys the entire backend API as a Serverless function.

🌐

Unified Origin Routing

Using the vercel.json rewrites, front-end assets at root are served statically while any `/api/*` paths automatically map to the Express runtime, bypassing CORS issues.

🔒

Secure Environment

Database credentials, JWT keys, and API secrets are securely configured inside the Vercel dashboard and injected at runtime.

vercel.json
{ "version": 2, "rewrites": [ { "source": "/api/:path*", "destination": "/backend/server.js" } ] }
